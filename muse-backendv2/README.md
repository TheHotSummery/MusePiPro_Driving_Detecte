# Muse Backend V2

Muse Pi Pro 疲劳驾驶监测系统的后端服务V2，基于 Spring Boot 2.6.13 开发，提供数据接收、存储、分析和展示功能。

请在 muse-backendv2\src\main\resources\application.properties  更新你的高德地图开发密钥


## 版本信息

- **当前版本**：v2.0
- **Spring Boot 版本**：2.6.13
- **Java 版本**：17
- **最后更新**：2025-12-10
